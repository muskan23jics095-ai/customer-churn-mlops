# Customer Churn Prediction Dashboard (MLOps)

This project predicts telecom customer churn using Machine Learning and Streamlit.

## Features
- Customer churn prediction
- Interactive dashboard
- Data visualization
- Model metrics
- ROC curve
- MLOps pipeline

## Tech Stack
Python, Scikit-learn, Streamlit, Pandas, NumPy

## Run Project

pip install -r requirements.txt
python src/train.py
streamlit run app/app.py
